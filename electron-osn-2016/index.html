What is electron?

  • A collection of open source technologies
  • An ecosystem of tools
  • An interface to native APIs
Electron is both a collection of open source technologies as well as an ecosystem of tools. When you think about Java, JavaScript or Node.js you can't discount the entire community of tools built up aroud it. That's electron in the abstract. More concretely electron provides access to native APIs.
What are the other competing projects NW.js (previously known as node-webkit), CEF (chromium embedded framework), Adobe AIR, Qt
Who's investing Github (Atom editor), Microsoft (VS Code), Facebook (Nuclide), Slack (Slack desktop), Basecamp (Basecamp desktop), Docker (Kitematic), Wordpress (Wordpress desktop)
